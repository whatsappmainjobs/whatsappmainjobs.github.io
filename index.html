  • • Identificador
  • • Mensaje
  • • (Incluye plantillas)

ℹ️ Información del Generador

✅ Datos que se rellenan automáticamente:

  • • FECHA: Se rellena en todas las filas con la fecha configurada
  • • HORA: Se rellena en todas las filas con la hora configurada
  • • NOMBRE WEBINAR: Se rellena en todas las filas con el nombre configurado
  • • URL: Se rellena en todas las filas con la URL configurada
  • • NOMBRE COMPLETO: Fórmula que combina NOMBRE + APELLIDOS
  • • TELEFONO FORMATEADO: Fórmula que añade el prefijo 34 si es necesario

⚠️ Instrucciones:

  1. Configura los datos del webinar arriba
  2. Selecciona cuántas filas quieres preparar con fórmulas
  3. Haz clic en "Generar Excel Plantilla"
  4. Abre el archivo en Excel y añade tus datos en la hoja "Identificacion"
  5. Ve a la sección "Envío de Mensajes" para usar el archivo
